The Code for RACECAR

                        
                            //get the user input from the page
                            //controller function
                            function getValue() {
                                //make sure alert is invisible
                                document.getElementById("alert").classList.add("invisible");
                            
                                let userText = document.getElementById("userText").value;
                            
                                //check for a palindrome
                                let returnObj = palindromeCheck(userText);
                            
                                //display message to the page
                                displayMessage(returnObj);
                            }
                            //check if text is a palindrome
                            function palindromeCheck(userText) {
                            
                                //racecar
                                userText = userText.toLowerCase();
                            
                                //remove spaces and special characters
                                let regex = /[^a-z0-9]/gi;
                                userText = userText.replace(regex, "");
                            
                                let revText = [];
                                let returnObj = {};
                            
                                //reverse a string using a for loop
                                for (let index = userText.length - 1; index >= 0; index--) {
                                    revText += userText[index];
                            
                                }
                            
                                if (revText == userText) {
                                    returnObj.msg = "Well Done! You Entered A Palindrome."
                                }
                                else {
                                    returnObj.msg = "Sorry! You Did Not Enter A Palindrome."
                                }
                            
                                returnObj.reversed = revText;
                                return returnObj;
                            
                            }
                            //display the message to the screen
                            function displayMessage(returnObj) {
                            
                                document.getElementById("alertHeader").innerHTML = returnObj.msg;
                                document.getElementById("msg").innerHTML = `Your phrase reversed is: ${returnObj.reversed}`;
                                document.getElementById("alert").classList.remove("invisible");
                            }

The Code is structured in three functions.

getValue

In this first function we start by initiating a process by which we get the inputted values from the screen so we can manipulate it and find out if it meets our programs requirements. In order to do this we use document.getElementById to specify the users text and assign it to the variable named just that: userText Next order of business is to sheild our beloved alert message from the page that notifies the user if they were successful or not. We do this by using the ever so useful document.getElementById method once more and adding to the "alert" id's classList and inputting "invisible" to make it remain unseen for the time being. Next we need to check if our users input in a palindrome or not, we make this happen in the next function palindromeCheck which has been assigned to another variable to be used as an object called returnObj so we can lastly attach it to our displayMessage function to display to the page once all requirements have been met.

palindromeCheck

Here is where we essentially check to see if what was inputted by our user is a palindrome, first we need to eliminate all possiblities of a potential error to occur when accepting said input. First we want to make what we accept case sensitive, specifically lower case by use of the .toLowerCase method which converts a string to lower case letters. We follow that up by also excluding any spaces and special characters by converting any of which that may be submitted to an empty string using our regex and .replace method. Now that we have that set up nicely, we will begin to set up our for loop to reverse our string and validate it is what we expect it to be in decrementing fashion. We instantiate an empty string called revText and also make our returnObj accept many possible properties by use of curly braces, two of which we will want it to hold is a message and a reversed string. In our loop we use our userText's length be our index, and as long that length is larger than the integer 0 we to decrement backwards until that is no longer the case, essentially starting from the end our inputted text retreating to the start of its index. In order to properly do this our indexes length need to be initiated with a minus one next to it because all indexes in arrays are zero based. Now we will take our revText variable and add to it as we iterate through the indexes of our userText and in turn that will reverse our string. To conclude this function we resort to an if else statement to verify if our reversed text equals or matches the original users text, and if it does we will relay a message saying if it does or not creating our returnObj.msg message.

displayMessage

This accepts returnObj as a parameter which is embodied with the palindromeCheck function. With a series of uses of document.getElementById we proceed to display our message to user. First we grab the "alertHeader" Id and attach our returnObj message in the innerHTML. Next we grab the "msg" id and make our own message using a template literal to input the value of returnObj.reversed into our statement. Lastly we have to unhide our "alert" from the page by doing just the opposite of what was done earlier when we added to its classList, now we are to simply remove from said classList to unveil our alert message to the page.
